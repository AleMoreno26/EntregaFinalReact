

# Mi eCommerce con React

## Descripción
Una aplicación de comercio electrónico construida con React que permite explorar productos, agregarlos al carrito y realizar compras.

## Características
- Listado de productos
- Detalle de productos
- Carrito de compras
- Gestión de pedidos



## Demo
Puedes ver una versión en vivo de la aplicación en [Netlify](https://tu-proyecto.netlify.app).


## Tecnologías Utilizadas
- React
- Bootstrap
- Firebase
- Vite
- Netlify


- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) utiliza [Babel](https://babeljs.io/) para Fast Refresh.
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) utiliza [SWC](https://swc.rs/) para Fast Refresh.
